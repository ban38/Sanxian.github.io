C++解題

1. 輸入一個數字，判斷是否為質數。

程式碼如下： 

            #include <iostream>
            using namespace std;
            int main() {
                int num;
                cout << "Enter a number: ";
                cin >> num;
                bool prime = true;
                for (int i = 2; i < num; i++) {
                    if (num % i == 0) {
                        prime = false;
                        break;
                    }
                }
                if (prime) {
                    cout << num << " is a prime number." << endl;
                } else {
                    cout << num << " is not a prime number." << endl;
                }
                return 0;
            }

這個程式碼會先讓使用者輸入一個數字，然後判斷是否為質數。判斷質數的過程是，先將數字除以 2 到數字的平方根，如果有餘數，則為非質數。如果沒有餘數，則為質數。

            using namespace std;

                int num;
                cout << "Enter a number: ";
                cin >> num;
                if (num % 2 == 1) {
                    cout << num << " is an odd number." << endl;
                } else {
                    cout << num << " is an even number." << endl;
                }
                return 0;
            }

這個程式碼會先讓使用者輸入一個數字，然後判斷是否為奇數。判斷奇數的過程是，只要數字除以 2 餘 1，就表示為奇數。
